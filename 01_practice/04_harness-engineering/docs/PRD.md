# PRD: CVE Insight

## 목표
쏟아지는 CVE 취약점을 빠르게 이해하고 대응할 수 있도록, CVE 번호·URL·뉴스 텍스트를 입력하면 취약점 분석·조치방안·유사 취약점·실습 시나리오까지 한 번에 제공하는 웹 도구.

## 사용자
- 보안 담당자 / 인프라 엔지니어: 새로 공개된 CVE의 영향도와 조치방안을 신속히 파악해야 하는 사람
- 보안 학습자: 취약점의 원리와 실습 시나리오를 통해 학습하려는 사람

## 핵심 기능
1. **입력 수집** — CVE 번호(예: CVE-2024-3094), 관련 URL, 뉴스 기사 텍스트 중 하나 이상을 입력받는다.
2. **데이터 보강** — CVE 번호가 있으면 NVD API로 공식 메타데이터(CVSS 점수, 영향 제품, 설명, 참조 링크)를 조회한다.
3. **LLM 분석** — NVD 메타데이터 + 사용자 입력을 OpenAI에 전달해 아래 4개 섹션을 생성한다:
   - 취약점 분석/설명 (무엇이, 어떻게, 왜 위험한지)
   - 조치방안 (패치, 완화책, 우선순위)
   - 유사/관련 취약점
   - 실습 시나리오 (안전한 학습용)
4. **결과 표시** — 4개 섹션을 구조화된 UI로 보여준다.

## MVP 제외 사항
- 분석 이력 저장 / DB (Stateless로 동작)
- 사용자 인증 / 계정
- CVE 자동 모니터링·알림(피드 구독)

## 디자인
- 다크모드 고정, 미니멀
- 무채색 기반 + 위험도(CVSS)에 따른 포인트 컬러 1가지(빨강 계열)
- 입력은 상단, 결과 4개 섹션은 카드/아코디언으로 정리
