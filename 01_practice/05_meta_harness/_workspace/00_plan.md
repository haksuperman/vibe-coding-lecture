# 제작 기획안: 커밋 가드 — 클로디 (Commit Guard: Claudy)

> 이전 "비 오는 날" 컨셉은 폐기됨. 3컷 소규모 1화를 전체 파이프라인으로 완성하는 기획안.

## 로그라인
야근하던 주니어 개발자가 API 키를 코드에 그대로 하드코딩한 채 푸시하려는 순간, 의인화된 AI 짝꿍 '클로디'가 키가 유출되기 직전에 낚아채 막아준다.

## 장르·톤·무드
- **장르**: 개발자 일상 · 보안 계몽 코미디 (의인화 마스코트물)
- **톤**: 밝고 따뜻한 코지(cozy) 코미디. 위협은 귀엽게, 교훈은 분명하게. 잔잔한 안도감으로 마무리.
- **시각 톤(무드)**: 늦은 밤 1인 작업실. 따뜻한 책상 스탠드 주황빛 + 모니터의 차가운 청색광 대비. 부드러운 셀셰이딩, 깔끔한 라인아트, 채도 중상. 클로디와 키도깨비는 살짝 발광(글로우)하는 반(半)디지털 존재로 처리해 현실 인물과 구분.

## 타깃 독자
- 20~30대 개발자 및 IT 종사자, 그리고 Claude Code/AI 코딩 툴에 관심 있는 입문자.
- "나도 저런 실수 했었지" 공감 + 보안 습관 환기를 동시에 노리는 계몽형 가벼운 읽을거리.

## 주요 캐릭터 (한 줄 + 시각 식별점)
- **도현 (남, 28, 주인공 주니어 개발자)** — 마감에 쫓겨 보안보다 '동작'이 먼저인 피곤한 개발자. 식별점: *헝클어진 검은 머리, 검은 뿔테 안경, 차콜 그레이 후드티, 진한 다크서클, 손에 든 별 무늬 머그컵.*
- **클로디 (Claude Code 의인화, AI 수호 마스코트)** — 도현의 터미널에 사는 작고 둥근 AI. 침착하고 다정하지만 보안 앞에선 단호. 식별점: *코랄·주황빛 둥근 몸체, 머리 위 한 가닥 안테나 끝의 빛나는 별표(✳) 심볼, 점 두 개 눈, 깜빡이는 언더스코어(_) 커서 입, 은은한 홀로그램 글로우.*
- **키도깨비 (시크릿 키 의인화, 장난꾸러기 위협)** — 하드코딩된 API 키가 깨어난 작은 도깨비. 공개 레포로 탈출해 세상에 퍼지려 한다. 식별점: *몸통이 황동 열쇠 모양, 빨간 작은 뿔, 음흉한 미소, 등에 'sk-live-...' 라벨 태그, 보라빛 글리치 잔상.*

## 전체 연재 구조 + 이번 1화 구성
**전체 구조(옴니버스 계몽 시리즈 가정):** 매 화가 하나의 보안 실수를 다루는 독립 에피소드. [1화] 하드코딩된 시크릿/API 키 → [2화] .env 유출·.gitignore 누락 → [3화] SQL 인젝션 → [4화] 의존성 취약점 등. 매 화 공통 포맷은 '개발자의 실수 → 위협 의인화 등장 → 클로디가 사전 차단 → 올바른 습관 제시'다.

**1화 구성(도입→절정→훅, 3컷):** 깊은 밤, 도현이 기능을 겨우 완성하고 `API_KEY = "sk-live-..."`를 코드에 박은 채 무심코 `git commit && git push`를 입력한다(도입, 컷1). 엔터를 누르는 순간 코드 속에서 키도깨비가 튀어나와 공개 레포 너머로 탈출하려 하고, 터미널에서 클로디가 빛을 내며 솟아올라 "잠깐, 그거 진짜 키예요!"라고 외치며 푸시를 가로막는다(절정·보안사고 포착, 컷2). 클로디가 키를 `.env`로 옮기고 `.gitignore`에 추가해 키도깨비를 안전하게 봉인하자 도현이 식은땀을 닦으며 안도한다 — 그때 클로디가 윙크하며 "그럼… 다음 녀석도 막아볼까요?" 화면 구석엔 다음 위협 'SQL 인젝션'의 그림자가 어른거린다(훅, 컷3).
