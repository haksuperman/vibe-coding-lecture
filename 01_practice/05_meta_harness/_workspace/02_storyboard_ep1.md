# 커밋 가드 — 클로디 1화 콘티

> 세로 스크롤 3컷. 컷 번호는 작화·QA가 동일 참조하는 영구 키.

## 컷 1 — 무심한 커밋 (도입)
- 등장인물: 도현
- 샷·앵글: 미디엄~바스트 샷, 약간의 어깨 너머(OTS) 앵글. 도현의 옆얼굴과 모니터를 한 화면에.
- 구도 설명: 화면 좌측에 후드티 입은 도현의 옆모습(하품, 다크서클), 우측에 듀얼 모니터. 모니터 중앙에 코드 한 줄 `API_KEY = "sk-live-..."`가 또렷하게 강조(살짝 확대/하이라이트). 책상 앞쪽에 별 무늬 머그컵. 따뜻한 스탠드 주황빛 + 모니터 청색광 대비. 터미널엔 `git commit -m "기능 완성" && git push` 타이핑 중, 엔터 위 손가락.
- 대사·SFX: 도현 말풍선 "휴… 드디어 됐다. 바로 올리고 자야지." / SFX: 작게 "타닥… 타닥" (타이핑).
- 연출 노트: 스크롤 시작 컷. 차분한 호흡. 독자 시선이 자연스럽게 코드의 `sk-live-...`에 멈추도록 그 줄만 미세하게 밝게. 위험을 글이 아닌 화면으로 심는다. 컷 아래 약간의 여백으로 '엔터 직전'의 정적.

## 컷 2 — 잠깐, 그거 진짜 키예요! (절정·임팩트 컷)
- 등장인물: 도현, 키도깨비, 클로디
- 샷·앵글: 크고 단독 배치되는 세로 임팩트 컷. 다이내믹 앵글(살짝 앙각), 화면 깊이감.
- 구도 설명: 화면 중앙, 코드 줄 사이에서 튀어나온 키도깨비(황동 열쇠 몸, 빨간 뿔, 등에 흔들리는 'sk-live-...' 라벨)가 우상단 'PUBLIC REPO'라 적힌 빛의 문으로 도망치려 함 — 보라빛 글리치 잔상. 그 앞을 코랄빛 글로우의 클로디(안테나 끝 별표 ✳)가 솟아올라 키도깨비를 붙잡아 가로막음. 우측 터미널 창에 빨간 경고 `Blocked: hardcoded secret detected (sk-live-...)`. 좌하단에 눈이 커진 도현의 놀란 얼굴(다크서클+뿔테 안경 유지).
- 대사·SFX: 클로디 "잠깐, 그거 진짜 키예요!" / 키도깨비 "히히, 밖으로 나갈 거지롱~" / 도현 "헉…!" / SFX: 강한 "철컥!" (붙잡는 소리), 경고음 "삐빅".
- 연출 노트: 1화 감정 절정. 가장 큰 세로 컷으로 단독 배치, 위아래 여백을 줄여 '쾅' 떨어지는 스크롤 충격. 빛의 문→클로디→도현으로 이어지는 대각선 시선 유도. 빨간 경고색이 코랄/주황 톤 위에서 한 번 튄다.

## 컷 3 — 봉인 그리고 다음 위협 (훅)
- 등장인물: 도현, 클로디, (실루엣) 다음 위협 SQL 인젝션
- 샷·앵글: 미디엄 투샷 + 화면 구석 인서트. 안정적인 정면~약한 부감.
- 구도 설명: 중앙에 자물쇠 채워진 `.env` 상자 안에서 잠잠해진 키도깨비, 그 위로 클로디가 한쪽 점눈을 찡긋(윙크)하며 떠 있음. 터미널엔 초록색 `Safe: key moved to .env, added to .gitignore`. 좌측에 식은땀 닦으며 안도하는 도현(어깨 풀린 자세). 우하단 어둠 속 인서트로 'SQL 인젝션' 라벨이 희미한 또 다른 그림자 눈동자 실루엣.
- 대사·SFX: 클로디 "키는 .env로 옮기고 .gitignore에 넣었어요. 이제 안전해요." → 도현 "고마워 클로디… 너 없었으면 큰일 날 뻔했다." → 클로디 "그럼… 다음 녀석도 막아볼까요?" / SFX: 잠금 "찰칵", 안도의 "후우…".
- 연출 노트: 절정 뒤 호흡을 늦춰 안도감. 밝은 초록 안전색으로 톤을 풀어준다. 마지막에 우하단 그림자 인서트를 작게 배치해 스크롤 끝에서 다음 화 궁금증(훅)을 남긴다. 그림자는 실루엣·라벨만, 설명 금지.
