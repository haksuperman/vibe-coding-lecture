# 캐릭터 시트: 커밋 가드 — 클로디

> 외형은 작화 프롬프트가 그대로 재사용한다. 추상 형용사가 아니라 시각적 명사로 적는다.

## 도현 (Dohyun)
- 외형: 28세 한국인 남성. 머리 — 헝클어진 짧은 검은 머리, 약간 뻗침. 눈 — 짙은 갈색, 눈 밑 진한 다크서클로 피곤한 인상. 검은 뿔테 안경. 체형 — 보통 키(175cm) 마른 편, 살짝 굽은 자세. 기본 의상 — 차콜 그레이 후드티(끈 달림), 무지 검은 티셔츠. 상징 소품 — 흰 별표(✳) 무늬가 그려진 머그컵. 식별 포인트 — 다크서클 + 검은 뿔테 안경 + 후드티.
- 성격: 성실하지만 마감에 쫓기면 '일단 동작'을 우선하는 조급함이 결함. 실수 후엔 솔직히 반성한다.
- 말투: 1인칭 "나/내". 반말 혼잣말 위주. 어미 "~네", "~잖아", "헉", "아 맞다". 피곤한 한숨 섞인 짧은 입말.
- 동기: 원하는 것 — 기능을 빨리 완성해 자고 싶다. 두려워하는 것 — 사고 치는 것, 키 유출로 인한 비용 폭탄·문책.
- 관계: 클로디를 의지하면서도 가끔 잔소리로 느낀다. 키도깨비는 자기가 무심코 깨운 화근.

## 클로디 (Claudy) — Claude Code 의인화
- 외형: 작고 둥근 마스코트(손바닥 크기). 몸체 — 따뜻한 코랄·주황빛 둥근 형태, 매끈한 무광 표면에 은은한 홀로그램 글로우. 머리 위 — 한 가닥 가는 안테나, 그 끝에 빛나는 별표(✳, 사방으로 뻗는 4~8갈래 선버스트) 심볼. 눈 — 단순한 점 두 개(까만 동그라미). 입 — 깜빡이는 터미널 커서 모양(언더스코어 _). 팔다리 — 짧고 둥근 스텁. 식별 포인트 — 코랄빛 둥근 몸 + 안테나 끝 별표 + 커서 입.
- 성격: 침착하고 다정한 조력자. 평소엔 부드럽지만 보안 위협 앞에서는 단호하고 빨라진다.
- 말투: 1인칭 "저". 정중한 존댓말. 어미 "~요", "~예요", "~볼까요?". 짧고 또렷한 경고. 말버릇 "잠깐," 으로 개입을 연다.
- 동기: 원하는 것 — 도현을 사고로부터 지킨다. 두려워하는 것 — 자신이 늦어 키가 이미 public으로 나가버리는 것(사전 차단 제약 때문).
- 관계: 도현의 터미널 짝꿍·수호자. 키도깨비와는 천적.

## 키도깨비 (Key-Gremlin) — 하드코딩된 시크릿 의인화
- 외형: 작은 도깨비(클로디보다 약간 작음). 몸통 — 광택 나는 황동 열쇠 모양(머리 부분이 얼굴, 톱니가 다리). 빨간 작은 뿔 두 개. 눈 — 가늘게 찢어진 음흉한 눈. 입 — 삐죽한 음흉한 미소(작은 송곳니). 등 — 'sk-live-...' 라벨 태그가 매달려 흔들림. 식별 포인트 — 황동 열쇠 몸 + 빨간 뿔 + 보라빛 글리치 잔상 + sk-live 라벨.
- 성격: 장난꾸러기 탈주범. 공개 레포로 빠져나가 세상에 퍼지려는 충동.
- 말투: 1인칭 "나". 까불대는 반말. 어미 "~지롱", "~다고~", 낄낄대는 웃음 "히히". 짧고 약 올리는 톤.
- 동기: 원하는 것 — public 레포 밖 인터넷으로 탈출. 두려워하는 것 — .env 봉인과 .gitignore 격리.
- 관계: 도현이 무심코 깨운 존재. 클로디에게 번번이 막힌다.
