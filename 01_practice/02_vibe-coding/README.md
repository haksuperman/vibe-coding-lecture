# 💳 카드 내역서 분석기

카드사에서 받은 **CSV / Excel(xls·xlsx) 내역서**를 올리면 어디에 얼마를 썼는지 분석해서 보여주는 웹 앱입니다.

- **완전 클라이언트 사이드** — 카드 데이터는 브라우저 안에서만 처리되고 어디에도 전송되지 않습니다.
- **오프라인 동작** — CSV 파서·차트는 직접 구현했고, Excel 파싱용 SheetJS(`xlsx.full.min.js`)도 폴더에 함께 두어 인터넷 없이 동작합니다.

## 실행 방법

로컬 HTTP 서버로 띄워서 사용하세요. (`file://`로 직접 열면 브라우저가 보안상
일부 리소스 로딩을 막아 `Unsafe attempt to load URL` 오류가 날 수 있습니다.)

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

처음이라면 화면의 **"샘플 데이터로 먼저 둘러보기"** 버튼으로 동작을 확인해 보세요.

## 입력 CSV 형식

**CSV와 Excel(.xls/.xlsx) 모두** 지원합니다. Excel은 첫 번째 시트를 읽고, 날짜 셀은 자동으로 인식합니다. 한국 카드사 일반형을 기준으로 컬럼명을 자동 인식합니다.

| 의미 | 인식하는 컬럼명 예시 |
|------|----------------------|
| 날짜 | 이용일자, 거래일자, 승인일자, 사용일자 … |
| 가맹점 | 가맹점명, 가맹점, 사용처, 적요 … |
| 금액 | 이용금액, 승인금액, 결제금액, 금액 … |
| 카테고리(선택) | 카테고리, 업종, 분류 … |

- 파일 앞부분에 안내문/요약 줄이 있어도 **실제 헤더 행을 자동으로 찾아냅니다.**
- **EUC-KR / UTF-8** 인코딩을 자동 감지합니다.
- 카테고리 컬럼이 없으면 가맹점명으로 **카테고리를 추정**합니다.
- 자동 인식이 틀리면 화면에서 컬럼을 **직접 지정**할 수 있습니다.

샘플은 [`sample_statement.csv`](./sample_statement.csv) 참고.

## 분석 내용

- 총 지출 / 거래 건수 / 분석 기간 / 일·월 평균 지출 / 가장 많이 쓴 날
- 월별 지출 추이
- 카테고리별 지출 비중
- 요일별 지출 패턴
- 가장 많이 쓴 가맹점 TOP 10
- 환불·취소(음수 금액)는 순지출에 반영

## 파일 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 |
| `styles.css` | 스타일 |

| `xlsx.full.min.js` | Excel 읽기용 SheetJS 라이브러리 (오프라인 동봉) |
| `sample_statement.csv` | 예시 데이터 |
